CS631 -- Advanced Programming in the UNIX Environment - Intro

Intro

Summary

One of the most important pre-requisites for this class -- the ability to work in a Unix-like environment -- cannot easily be measured by students having completed a given previous class.

To ensure that students are able to perform even the most basic of tasks required to follow the course in the required environment, you are asked to run the script cs631-intro.sh on linux-lab.cs.stevens.edu.

That is:

  • log in on linux-lab.cs.stevens.edu
  • fetch the file http://www.cs.stevens.edu/~jschauma/631/cs631-intro.sh
  • run the script

The due date for this assignment is 2014-09-15 16:00. (You do not need to submit anything -- the script will do everything necessary on your behalf.)
