CS631 -- Advanced Programming in the UNIX Environment - HW#3

HW#3: webserver framework

Summary

The objective of this assignment is for you to lay down the basic framework and HTTP functionality for your final project. This ensures that your final project will be on track.

In particular, you are to write the following portions of your final project:

  • the main function, parsing all command-line options and setting all flags
  • the basic code to listen on the given network address and port, accept connections
  • basic HTTP protocol validation
Your rudimentary server will read one line of text from any connecting client and validate that the input is a proper HTTP request. It will then return the correct HTTP code for the following use cases:
  • input was a valid HTTP request supported by this server => 200 OK
  • input was a valid HTTP request not supported by this server => 501 Not Implemented
  • input was a valid HTTP request of an HTTP version not supported by this server => 505 Version Not Supported
  • input was an invalid request => 400 Bad Request
Your server should also time out open client connections after a suitable amount of time, yielding a 408 or 522 Connection Timed Out status.

Deliverables and Due Date

You will submit at least three files:

  • main.c - containing the basic functionality
  • net.c - containig any network related functionality
  • http.c - containing HTTP specific functionality
